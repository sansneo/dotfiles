# dotfiles
> For a comfy and distraction-free setup!

My additions to the [iso](https://github.com/sansneo/iso) xorg build.

1. Run `install.sh`
2. Install my dotfiles
3. Install `codium`, and `thorium`, then create symbolic links for them to somewhere included in your $PATH
4. Add the Bitwarden, YouTube Unhook and CouponBirds(serious business here) extensions to `thorium` and add the uBlock Origin filters
5. Add the GitHub Themes, Vim, Tinymist Typst and Go extensions to `codium`
6. Done! 🎉

# Workflow
As usual, just like in [iso](https://github.com/sansneo/iso), we go for pragmatism and understanding the system deeply!  
Using a smaller selection of quality tools I master over time and am not afraid of building muscle memory for,  even if it means a short term speed loss for a greater future productivity boost.  
I wanna be cracked at programming, ship and excel; never mediocre; this is the way for everything I do personally in life!  

Each workspace has a function and _usually_ has a single application within it!  
Kind of like The Primeagen workflow. Just more optimized! *wink wink*

I experimented with a lot of different workflows.  
I'm always learning and changing things, but this is such a good spot to be in!  

Also, this is inspired by [Sylvan Franklin](https://www.youtube.com/@sylvanfranklin) which is really cool and talented.
